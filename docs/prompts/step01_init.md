# Step 1: GASプロジェクトの初期生成

## 目的
`docs/01_requirements.md` に記載された要件を満たす初期のGASコード（バックエンドおよびフロントエンド）を `src/` ディレクトリに生成する。

## タスク
1. `docs/01_requirements.md` の要件を読み込むこと。
2. 以下の3つのファイルを `src/` 内に生成すること。
   - `src/env.gs`: スプレッドシートIDなどの定数を定義するファイル（プレースホルダーのみ記述し、実際の値は入れないこと）。
   - `src/コード.gs`: `env.gs` で定義されたIDを読み込み、スプレッドシートからデータを取得してJSONで返すAPI機能と、HTML配信ロジックを実装。
   - `src/index.html`: Tailwind CSSとVue.js(またはAlpine.js)をCDNで読み込み、全体俯瞰マトリクスと月別チェックリストを描画・印刷（`@media print`）する初期UIを実装。
3. コード生成後、ターミナル上に以下の手順を簡潔に出力すること。
   - ローカルの `src/env.gs` に実際のスプレッドシートIDを設定する方法
   - コードをGAS環境に反映（デプロイ）する方法

## 注意事項
リポジトリのルール（`CLAUDE.md`）に従い、`コード.gs` や `index.html` にスプレッドシートID等のシークレットをハードコードしないこと。
